# Documentation Index

## Manual

- `manual/` — the task-oriented **user manual**: concepts, end-to-end journeys, and
  auto-generated CLI / `lake.*` API / canonical-table reference. Start at
  `manual/index.md`. Reference pages are generated by
  `scripts/gen_docs_reference.py` and guarded by
  `tests/test_docs_reference_current.py`; narrative chapters are kept current via
  the `doc-sync` skill. Preview with `mkdocs serve` (config in `mkdocs.yml`).
- `product/documentation-manual-epic.md` — the top-down information architecture
  for the full manual (built from the whole backlog) and its decomposition into
  backlog items 0289–0296; the plan for backfilling the manual chapter by chapter.
